# Ubuntu WCLI nginx php7 gearman supervisor

What does this image contain?

- Ubuntu:16.04
- nginx:latest
- WPCLI:latest
- php7.0x and php7.0x-fpm
- gearman
- supervisor

## Contributing

Run: 
```$bash
npm install
```
Make sure the lint-ers pass.
